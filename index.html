Decision-time anchoring converts BPTT marginal values into optimal controls for multiplicative kernels and equilibrium controls otherwise.

Model and objective

$$dX_t = X_t\bigl(r + \pi_t(\mu-r)\bigr)dt + X_t\pi_t\sigma\,dW_t - c_t\,dt$$
$$U(x)=\frac{x^{1-\gamma}}{1-\gamma},\qquad \max_{(\pi,c)} \mathbb{E}\left[U(X_T)\right]$$

Hamiltonian

$$H(x,\pi,\lambda)=\lambda x\bigl(r+\pi(\mu-r)\bigr)-\frac{1}{2}\gamma\lambda x\sigma^2\pi^2$$

Recovered control

$$\hat{\pi}_t=-\frac{\hat{\lambda}_t}{X_t\,\partial_x\hat{\lambda}_t}\,\frac{\mu-r}{\sigma^2}$$

State and liquidation wealth

$$Z_t=(X_t,Y_t),\qquad L_t=X_t+(1-\alpha)Y_t$$
$$U(\ell)=\frac{\ell^{1-\gamma}}{1-\gamma},\qquad \max_u\mathbb{E}\left[U(L_T)\right]$$

Nonsmooth local Hamiltonian

$$H(u;\lambda)=L_t\left[-\lambda_xu^+ +(1-\alpha)\lambda_xu^-+\lambda_yu\right]$$
$$H^{\varepsilon}(u;\lambda)=H(u;\lambda)-\frac{\varepsilon}{2}L_t\lambda_xu^2$$

Dead-zone projection

$$R=\frac{\lambda_y}{\lambda_x},\qquad P_\varepsilon(R)=\begin{cases} \dfrac{R-1}{\varepsilon},&R>1,\\[2pt] 0,&1-\alpha\le R\le1,\\[2pt] \dfrac{R-(1-\alpha)}{\varepsilon},&R<1-\alpha. \end{cases}$$

Pointwise admissible set

$$U(t,x)=\{(\pi,C):\mathbf 1^\top\pi=1,\ \Gamma(t,x;\pi,C)\ge0\}$$
$$dX_t=(X_t\pi_t^\top\mu^e-C_t)dt+X_t\pi_t^\top v^e dW_t$$

Generalized Hamiltonian

$$\mathcal H=e^{-\rho t}U(C)+\lambda b+\zeta^\top\sigma+\frac12P\|\sigma\|^2$$
$$\hat\lambda\approx\mathbb E[D_XJ],\qquad \hat P\approx\mathbb E[D^2_{XX}J]$$

Local KKT / QP recovery

$$\hat u=\arg\max_{u\in K}\left\{a^\top u-\frac12u^\top Qu\right\}$$
$$a=x\{\hat\lambda(\mu-r\mathbf1)+v\hat\zeta\},\qquad Q=-x^2\hat P\Sigma$$

Anchored objective

$$J(t_0,x;u)=\mathbb E_{t_0,x}\!\left[\int_{t_0}^{T}D(t_0,s)\ell(s,X_s,u_s)ds+D(t_0,T)g(X_T)\right]$$

Anchored Hamiltonian

$$H(t_0,t,x,u,\lambda,Z)=D(t_0,t)\ell(t,x,u)+\langle\lambda,b\rangle+\operatorname{Tr}(Z^\top\sigma)$$

Diagonal local correction

$$\hat u(t,x)\in\arg\max_{u\in U(x)}H\bigl(t,t,x,u,\hat\lambda(t,x),\hat Z(t,x)\bigr)$$
$$\text{multiplicative }D:\ \text{optimal};\qquad \text{otherwise}:\ \text{equilibrium}$$

Core examples. Smooth Merton and transaction costs share the rollout and costate panels; only the local costate-to-action map changes.

Scope. The transaction-cost view uses representative sell, hold, and buy query points. Displayed convergence curves are schematic rather than live training output.

Constrained portfolios. Activations enforce pointwise control feasibility; first- and second-order adjoints then define a solver-neutral local PMP/KKT recovery problem.

Scope. The visualization covers pointwise control constraints, including state-dependent control bounds. Genuine state-path constraints require additional reflected, nonsmooth, or singular-control machinery.

Beyond Bellman recursion. Each query is anchored at its decision time, BPTT estimates the corresponding adjoint, and a diagonal Hamiltonian solve synthesizes the action.

Scope. Multiplicative kernels lead to classical optimal controls; non-multiplicative kernels lead to time-consistent equilibrium controls. Curves are schematic educational outputs.
